# Give Me Back My Eyes!

Resource pack that brings proper eyes to those they were taken from!  
Now baby **pigs, cows, cats, wolves and fox** have proper eyes as in pre-26.1 versions!

## But I like new baby variants' eyes!
Then don't use this resource pack! It's purely **subjective** and I **subjectively** don't like new baby eyes, so I made this RP!
